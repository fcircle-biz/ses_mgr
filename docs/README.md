## ドキュメントの更新について

1. 各ドキュメントは Git で管理されています
2. 各ガイドラインに従ってドキュメントを作成・更新してください
3. 更新する際は必ず最新の内容を pull してから編集してください

```bash
# 最新版の取得
git pull origin main

# 変更のコミット
git add 変更したファイル
git commit -m "変更内容の説明"
git push origin main
```
