SES業務システム 共通画面設計

共通画面一覧

SES業務システムの共通画面設計に関する詳細です。各画面の詳細仕様は、画面IDのリンクから確認できます。

  • CMN-001: ログイン画面
    システムへのログイン認証を行う画面
  • CMN-002: ダッシュボード
    ユーザーごとのホーム画面・通知一覧を表示する画面
  • CMN-003: メニュー（グローバルナビゲーション）
    全画面で共通して表示されるグローバルナビゲーションメニュー
  • CMN-004: ユーザープロフィール画面
    ユーザー自身のプロフィール情報を参照・更新するための画面
  • CMN-005: 通知一覧画面
    ユーザー宛てのシステム通知・アラートを一覧表示し、既読/未読管理を行う画面
