SES業務システム 技術者管理画面設計

ENG-001: 技術者一覧画面

画面概要

項目 内容
画面ID ENG-001
画面名 技術者一覧
概要 全技術者の一覧表示・検索・フィルタリングを行う画面
対象ユーザー 人事担当者、営業担当者
前画面 CMN-002: ダッシュボード
次画面 ENG-003: 技術者詳細、ENG-002: 技術者登録

画面レイアウト

[技術者一覧画面のワイヤーフレーム]

画面構成要素

1. 検索・フィルター部分

項目ID 項目名 種類 説明
eng_list_001 検索ボックス テキスト入力 技術者名、技術者ID、スキルなどでのフリーワード検索
eng_list_002 稼働状況 セレクトボックス（複数選択可） 稼働状況によるフィルタリング（稼働中、待機中、予約済み、休職中など）
eng_list_003 スキル マルチセレクト スキルによるフィルタリング
eng_list_004 経験年数 レンジスライダー 経験年数範囲によるフィルタリング
eng_list_005 所属区分 セレクトボックス 所属区分によるフィルタリング（社員、協力会社など）
eng_list_006 最終更新日 日付範囲選択 プロフィール更新日によるフィルタリング
eng_list_007 検索ボタン ボタン 検索条件を適用して結果を表示
eng_list_008 クリアボタン ボタン 検索条件をクリア

2. 一覧表示部分

項目ID 項目名 種類 説明
eng_list_009 技術者一覧テーブル テーブル 検索条件に合致する技術者の一覧
eng_list_010 技術者ID テキスト（リンク） システム内部で管理される技術者の一意識別子。クリックで詳細画面へ
eng_list_011 氏名 テキスト（リンク） 技術者の氏名。クリックで詳細画面へ
eng_list_012 所属区分 テキスト 技術者の所属区分（社員、協力会社など）
eng_list_013 稼働状況 ラベル（色分け表示） 技術者の現在の稼働状況
eng_list_014 主要スキル タグリスト 技術者の主要スキル（最大5つ）
eng_list_015 経験年数 数値 技術者の経験年数
eng_list_016 対応可能日 日付 案件にアサイン可能となる日付
eng_list_017 アクションボタン ボタングループ 詳細表示、スキルシート、マッチング検索などの操作ボタン
eng_list_018 ページネーション ページ制御 複数ページにわたる結果のナビゲーション

3. 操作ボタン

項目ID 項目名 種類 説明
eng_list_019 新規技術者登録ボタン プライマリーボタン 新規技術者登録画面への遷移
eng_list_020 CSVエクスポートボタン セカンダリーボタン 表示中の技術者一覧をCSV形式でエクスポート
eng_list_021 CSVインポートボタン セカンダリーボタン CSV形式の技術者情報をインポート

画面動作仕様

  1. 画面初期表示時は、デフォルトですべての技術者を表示する
  2. 検索条件を指定して検索ボタンをクリックすると、条件に合致する技術者が一覧表示される
  3. 一覧の技術者ID・氏名をクリックすると、該当の技術者詳細画面（ENG-003）に遷移する
  4. アクションボタンの「詳細」をクリックすると、該当の技術者詳細画面（ENG-003）に遷移する
  5. アクションボタンの「スキルシート」をクリックすると、該当の技術者のスキルシート管理画面（ENG-004）に遷移する
  6. アクションボタンの「マッチング」をクリックすると、該当技術者とのマッチング検索画面（MTH-001）に遷移する
  7. 「新規技術者登録」ボタンをクリックすると、技術者登録画面（ENG-002）に遷移する
  8. 「CSVエクスポート」ボタンをクリックすると、現在の検索結果がCSVファイルとしてダウンロードされる
  9. 「CSVインポート」ボタンをクリックすると、CSVインポート用のモーダルダイアログが表示される
  10. ページネーションを使用して、複数ページにわたる検索結果をナビゲートできる

注意： ユーザーの権限レベルによって、表示される操作ボタンや実行可能な操作が異なります。

  • 人事担当者：全技術者の閲覧・編集が可能
  • 営業担当者：全技術者の閲覧のみ可能、編集は不可

ENG-002: 技術者登録画面

画面概要

項目 内容
画面ID ENG-002
画面名 技術者登録
概要 新規技術者情報を登録する画面
対象ユーザー 人事担当者
前画面 ENG-001: 技術者一覧
次画面 ENG-003: 技術者詳細（登録成功時）、ENG-001: 技術者一覧（キャンセル時）

画面レイアウト

[技術者登録画面のワイヤーフレーム]

画面項目定義

項目ID 項目名 種類 入力制限 必須 説明
eng_reg_001 氏名 テキスト入力 100文字以内 ◯ 技術者の氏名
eng_reg_002 氏名（カナ） テキスト入力 100文字以内、カタカナのみ ◯ 技術者の氏名（カタカナ）
eng_reg_003 所属区分 セレクトボックス - ◯ 技術者の所属区分（社員、協力会社など）
eng_reg_004 所属会社 テキスト入力 100文字以内 △ 協力会社の場合に入力
eng_reg_005 メールアドレス テキスト入力 有効なメールアドレス形式 ◯ 連絡用メールアドレス
eng_reg_006 電話番号 テキスト入力 有効な電話番号形式 ◯ 連絡用電話番号
eng_reg_007 生年月日 日付選択 有効な日付 ◯ 技術者の生年月日
eng_reg_008 性別 ラジオボタン - ◯ 技術者の性別
eng_reg_009 稼働状況 セレクトボックス - ◯ 現在の稼働状況
eng_reg_010 対応可能日 日付選択 有効な日付 △ 案件にアサイン可能となる日付
eng_reg_011 経験年数 数値入力 整数（0〜50） ◯ IT業界での経験年数
eng_reg_012 希望単価（下限） 数値入力 整数（0〜999999） ◯ 希望する単価の下限
eng_reg_013 希望単価（上限） 数値入力 整数（下限以上、999999以下） ◯ 希望する単価の上限
eng_reg_014 最寄り駅 テキスト入力 100文字以内 ◯ 居住地の最寄り駅
eng_reg_015 稼働可能エリア チェックボックスグループ - ◯ 稼働可能な地域
eng_reg_016 リモート対応 セレクトボックス - ◯ リモート勤務の可否
eng_reg_017 技術スキル マルチセレクト - ◯ 保有する技術スキル
eng_reg_018 資格 マルチセレクト - × 保有する資格
eng_reg_019 自己PR テキストエリア 1000文字以内 × 技術者の自己PR
eng_reg_020 備考 テキストエリア 1000文字以内 × その他の補足情報
eng_reg_021 登録ボタン ボタン - - 技術者情報の登録を実行
eng_reg_022 キャンセルボタン ボタン - - 入力をキャンセルして前画面に戻る

画面動作仕様

  1. 必須項目が入力されているかをバリデーションチェックする
    • 未入力の必須項目がある場合はエラーメッセージを表示
    • 入力形式や値の範囲が不正な場合はエラーメッセージを表示
  2. 所属区分で「社員」以外を選択した場合、所属会社の入力欄を必須として表示する
  3. 稼働状況で「稼働中」を選択した場合、対応可能日の入力欄は非表示とする
  4. 希望単価の下限が上限を超える値に設定された場合、エラーメッセージを表示する
  5. 「登録」ボタンをクリックした場合
    • 入力内容のバリデーションを行い、問題がなければ技術者情報を登録する
    • 登録成功時は技術者詳細画面（ENG-003）に遷移する
    • 登録失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  6. 「キャンセル」ボタンをクリックした場合
    • 入力内容を破棄して技術者一覧画面（ENG-001）に戻る
    • 入力内容がある場合は確認ダイアログを表示し、ユーザーの意思を確認する

ENG-003: 技術者詳細画面

画面概要

項目 内容
画面ID ENG-003
画面名 技術者詳細
概要 技術者の詳細情報表示・編集を行う画面
対象ユーザー 人事担当者、本人
前画面 ENG-001: 技術者一覧、ENG-002: 技術者登録
次画面 ENG-001: 技術者一覧、ENG-004: スキルシート管理、MTH-001: マッチング検索

画面レイアウト

[技術者詳細画面のワイヤーフレーム]

画面モード

この画面は以下の2つのモードを持ちます：

  1. 表示モード: 技術者の詳細情報を参照するためのモード（デフォルト）
  2. 編集モード: 技術者の詳細情報を編集するためのモード（人事担当者または本人のみ）

タブ構成

技術者詳細画面は以下のタブで構成されます：

  1. 基本情報: 技術者の基本的な情報（氏名、連絡先、状況など）
  2. スキル情報: 技術スキル、資格、経験など
  3. 稼働履歴: 過去の案件履歴
  4. マッチング履歴: 過去のマッチング履歴

主要な機能

  • 技術者の基本情報の表示・編集
  • 技術者のスキル情報の表示・編集
  • 稼働状況の変更
  • スキルシート管理画面への遷移
  • 過去の稼働履歴・案件履歴の表示
  • マッチング検索の実行

画面動作仕様

  1. 画面初期表示時は、表示モードで基本情報タブが選択された状態で表示
  2. 各タブをクリックすると、対応する情報が表示されるようタブ切り替えが行われる
  3. 「編集」ボタンをクリックすると編集モードに切り替わる（権限のある場合のみ表示）
  4. 編集モードでの「保存」ボタンクリック時
    • 入力内容のバリデーションを行い、問題がなければ技術者情報を更新する
    • 更新成功時は表示モードに戻り、更新された情報が表示される
    • 更新失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  5. 「キャンセル」ボタンをクリックすると、編集内容を破棄して表示モードに戻る
  6. 「スキルシート」ボタンをクリックすると、スキルシート管理画面（ENG-004）に遷移する
  7. 「マッチング検索」ボタンをクリックすると、この技術者の条件でマッチング検索画面（MTH-001）に遷移する
  8. 「一覧に戻る」ボタンをクリックすると、技術者一覧画面（ENG-001）に戻る

注意： 編集モードへの切り替えボタンは、編集権限を持つユーザー（人事担当者または本人）にのみ表示されます。

[スキルシート管理画面のワイヤーフレーム]

  • スキルシートのプレビュー表示
  • スキルシートの内容編集
  • スキルシートのPDF/Word形式での出力

  2. テンプレート選択ドロップダウンから別のテンプレートを選択すると、選択したテンプレートに応じたプレビューが表示される

画面概要

項目 内容
画面ID ENG-005
画面名 稼働状況管理
概要 技術者の稼働状況をカレンダー形式で管理する画面
対象ユーザー 人事担当者、営業担当者
前画面 ENG-001: 技術者一覧
次画面 ENG-001: 技術者一覧

画面レイアウト

[稼働状況管理画面のワイヤーフレーム]

主要な機能

  • 稼働状況のカレンダー表示（月次・週次・日次）
  • 技術者のフィルタリング（所属、スキル、稼働状況など）
  • 稼働状況の更新（案件アサイン、予約など）
  • 稼働予測レポートの表示

画面動作仕様

  1. 画面初期表示時は、現在の月のカレンダーで全技術者の稼働状況を表示
  2. カレンダービューの切り替えボタン（月次・週次・日次）で表示形式を変更
  3. 期間選択で表示するカレンダーの期間を変更
  4. 技術者フィルタで表示する技術者を絞り込み
  5. カレンダー内の各セルをクリックすると、対応する日付の稼働詳細情報を表示
  6. 稼働状況を変更する場合、対象の日付/技術者をクリックして詳細ダイアログを表示
    • 案件へのアサイン、予約、休暇などのステータスを設定
    • 関連する案件・顧客情報を選択
    • 備考情報を入力
    • 設定を保存すると、カレンダー表示に反映される
  7. 「稼働予測レポート」ボタンをクリックすると、選択期間の稼働予測情報がレポート形式で表示される
  8. 「一覧に戻る」ボタンをクリックすると、技術者一覧画面（ENG-001）に戻る
