勤怠入力画面

画面概要

画面ID: TMS-001
画面名: 勤怠入力画面
対象ユーザー: 技術者
前画面: ダッシュボード、カレンダー画面
次画面: 工数入力画面（TMS-002）

勤怠入力画面は、技術者が日々の勤務状況（出勤・退勤時間、休憩時間、勤務場所など）を記録するための画面です。 カレンダーUI上で日付を選択し、その日の勤怠情報を入力します。また、一定期間の勤怠状況をまとめて確認することも可能です。 この画面で入力されたデータは、勤怠管理および工数管理の基礎データとなります。

画面レイアウト

[勤怠入力画面のワイヤーフレーム表示位置]

カレンダー表示部分（月表示）、日次勤怠入力フォーム、勤怠サマリー表示部分から構成される画面

画面は主に以下の3つのエリアで構成されます：

  1. ヘッダー部分：年月選択、月切替ボタン、勤怠状況サマリー表示
  2. カレンダー部分：月単位のカレンダー表示、日ごとの勤怠状況を色分け表示
  3. 勤怠入力フォーム：選択した日付の詳細な勤怠情報を入力するフォーム

画面項目定義

1. ヘッダー部分

項目ID 項目名 種類 入力制限 必須 説明
H01 年月選択 ドロップダウン 過去6ヶ月～将来1ヶ月 必須 表示する年月を選択
H02 前月ボタン ボタン - - 前月のカレンダーを表示
H03 翌月ボタン ボタン - - 翌月のカレンダーを表示
H04 当月サマリー テキスト表示 - - 当月の出勤日数/予定日数、総労働時間、残業時間の表示
H05 月次勤怠提出ボタン ボタン - - 当月の勤怠データを提出する
H06 ステータス表示 アイコン+テキスト - - 月次勤怠データの状態（作成中/提出済/承認済など）

2. カレンダー部分

項目ID 項目名 種類 入力制限 必須 説明
C01 曜日ヘッダー テキスト表示 - - 日～土の曜日表示
C02 日付セル クリック可能要素 - - 各日付を表示、クリックで詳細入力
C03 勤怠状況アイコン アイコン表示 - - 出勤/休暇/欠勤などの状況をアイコンで表示
C04 勤務時間表示 テキスト表示 - - 各日の労働時間を簡易表示

3. 勤怠入力フォーム

項目ID 項目名 種類 入力制限 必須 説明
F01 日付表示 テキスト表示 - - 選択された日付と曜日を表示
F02 勤怠区分 ラジオボタン 出勤/欠勤/有給休暇/半休/特別休暇/リモート勤務 必須 勤怠の種類を選択
F03 開始時間 時刻入力 HH:MM形式 任意 勤務開始時刻（勤怠区分が出勤/リモート勤務の場合に表示）
F04 終了時間 時刻入力 HH:MM形式 任意 勤務終了時刻（勤怠区分が出勤/リモート勤務の場合に表示）
F05 休憩時間 数値入力 0.0～9.9時間、0.1単位 必須 休憩した時間の合計（勤怠区分が出勤/リモート勤務の場合に表示）
F06 勤務場所区分 セレクトボックス 客先/社内/リモート/その他 必須 勤務を行った場所の種類（勤怠区分が出勤/リモート勤務の場合に表示）
F07 勤務場所詳細 テキスト入力 100文字以内 任意 勤務場所の具体的な名称や住所（勤怠区分が出勤の場合に表示）

F09 残業時間 計算表示 - - 実働時間から規定労働時間を引いた時間（自動計算）
F10 深夜勤務時間 計算表示 - - 22:00～翌5:00の間に働いた時間（自動計算）
F11 備考 テキストエリア 200文字以内 任意 勤怠に関する特記事項

F13 工数入力へボタン ボタン - - 該当日の工数入力画面へ遷移
F14 キャンセルボタン ボタン - - 入力内容をリセットする

画面動作仕様

1. 初期表示

  • 初期表示時は現在の年月のカレンダーを表示
  • 既に勤怠データが入力されている日付は、カレンダー上でステータスに応じた色分け表示
  • 休日・祝日は背景色を変えて表示（休日：薄い赤、祝日：薄いオレンジなど）
  • 勤怠入力フォームは非表示

2. 日付選択時

  • カレンダー上の日付をクリックすると、その日付が選択状態になり、下部に勤怠入力フォームを表示
  • 既に入力済みのデータがある場合は、それをフォームに読み込む
  • 選択された日付が休日・祝日の場合は、その旨を表示
  • 選択された日が未来の日付の場合、入力可能だが、「未来の勤怠データを入力しています」という警告を表示

3. 勤怠区分変更時

  • 「出勤」または「リモート勤務」を選択した場合：開始時間、終了時間、休憩時間、勤務場所区分の入力欄を有効化
  • 「欠勤」「有給休暇」「特別休暇」を選択した場合：開始・終了時間等の入力欄を無効化し、日当たりの規定時間を自動入力
  • 「半休」を選択した場合：半日分の規定時間を自動入力し、午前/午後の選択肢を表示

4. 時間入力時

  • 開始時間と終了時間が入力されると、休憩時間を考慮して実働時間を自動計算
  • 実働時間が規定時間（例：8時間）を超える場合、超過分を残業時間として自動計算
  • 22:00～翌5:00の間に働いた時間がある場合、その時間を深夜勤務時間として自動計算
  • 実働時間が異常に長い場合（例：16時間以上）、警告メッセージを表示

5. 保存時

  • 必須項目が未入力の場合、エラーメッセージを表示して保存を中止
  • 入力内容に問題がなければ、データを保存し、成功メッセージを表示
  • 保存後、カレンダー表示を更新して、入力済み日付の表示を変更
  • 月次サマリー情報も更新

6. 月次勤怠提出時

  • 月次勤怠提出ボタンをクリックすると、確認ダイアログを表示
  • 確認後、月次勤怠データのステータスを「提出済」に変更
  • 提出後は、承認されるまで当月のデータ編集を制限（編集には特別権限が必要）

特記事項・注意点

  • 勤怠データは日付ごとに保存され、月単位で集計・管理される
  • 過去の承認済みデータは原則として編集不可（修正が必要な場合は管理者による操作が必要）
  • 工数入力へボタンで、当日の工数入力画面に遷移する際、勤怠情報が先に保存されていることが前提
  • リマインダー機能により、未入力の勤怠データがある場合、ログイン時に通知
  • モバイル対応により、スマートフォンからの入力も可能
  • プロジェクト別の勤務設定（標準勤務時間など）に応じて、残業時間の計算ルールが変わる場合がある
