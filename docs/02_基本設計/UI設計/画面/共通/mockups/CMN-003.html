メニュー（グローバルナビゲーション）

左側に表示されているのがグローバルナビゲーションメニューです。このメニューは全画面で共通して表示され、システム内の各機能に素早くアクセスできます。

以下の特徴があります:

  • トップレベルのメニューとサブメニュー構成
  • アクセス権限に応じた表示制御
  • レスポンシブデザイン（モバイル対応）
  • 視覚的フィードバック（アクティブ状態、ホバー効果）

このモックアップでは実際の動作を確認できます。メニュー項目をクリックすると展開/折りたたみされます。
