CMN-003: メニュー（グローバルナビゲーション）

画面概要

項目 内容
画面ID CMN-003
画面名 メニュー（グローバルナビゲーション）
概要 全画面で共通して表示されるグローバルナビゲーションメニュー
対象ユーザー 全ユーザー
前画面 ログイン画面
次画面 各機能画面への遷移

画面レイアウト

画面構成要素

1. ヘッダー部分

コンポーネント 説明 機能・振る舞い
システムロゴ 左上に表示されるシステム名 クリックするとダッシュボードに戻る
ハンバーガーアイコン モバイル表示時のメニュー開閉ボタン クリックでメニュー表示/非表示を切り替え
通知アイコン 未読通知を示すアイコンとバッジ クリックで通知一覧をドロップダウン表示
ユーザー情報 ユーザーアイコンと名前 クリックでユーザーメニューをドロップダウン表示

2. サイドメニュー部分

コンポーネント 説明 機能・振る舞い
トップレベルメニュー 主要機能へのリンク
  • 展開可能なメニューの場合、クリックで展開/折りたたみ
  • 現在のページに対応するメニューがハイライト表示
サブメニュー トップレベルメニュー配下の機能へのリンク
  • 親メニュー展開時に表示
  • クリックで対応する機能画面へ遷移
メニューアイコン 各メニュー項目の左側に表示されるアイコン 機能カテゴリを視覚的に識別するためのアイコン

2. レスポンシブ対応

画面サイズ 表示形式 動作
デスクトップ（992px以上） 左側固定の垂直ナビゲーション 常に表示、幅280px
タブレット（768px～991px） 折りたたみ可能な垂直ナビゲーション 初期状態では非表示、ハンバーガーメニューで表示切替
モバイル（767px以下） オーバーレイ方式のドロワーメニュー 画面上に重なる形で表示、背景タップで閉じる

3. ユーザーインタラクション

操作 振る舞い
メインメニュークリック サブメニューの展開/折りたたみ
サブメニュークリック 対応する機能画面へ遷移
ハンバーガーアイコンクリック モバイル表示時にメニューを表示/非表示
メニュー外クリック モバイル表示時にメニューを閉じる

注意： メニューの表示項目はユーザーの権限（ロール）によって動的に制御されます。ユーザーは自分がアクセス権を持つ機能のみがメニューに表示されます。権限のない機能へのアクセスを試みた場合は、権限エラーページにリダイレクトされます。メニューの状態（展開/折りたたみ）はローカルストレージに保存され、次回アクセス時に復元されます。
