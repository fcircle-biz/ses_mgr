CMN-002: ダッシュボード画面

画面概要

項目 内容
画面ID CMN-002
画面名 ダッシュボード
概要 ユーザーごとのホーム画面・通知一覧を表示する画面
対象ユーザー 全ユーザー
前画面 CMN-001: ログイン画面
次画面 各機能メニューへの遷移

画面レイアウト

画面構成要素

1. ヘッダー

コンポーネント 説明 機能・振る舞い
システムロゴ 左上に表示されるシステムロゴ クリックするとダッシュボードに戻る
通知アイコン 未読通知数を表示するアイコン クリックするとドロップダウンで通知一覧を表示
ユーザー情報 ログインユーザー名を表示 クリックするとプロフィール・設定・ログアウトメニューを表示

2. サイドメニュー

コンポーネント 説明 機能・振る舞い
機能メニュー システムの主要機能へのリンク一覧 各機能画面への遷移
未処理タスク ユーザーの未処理タスク一覧 クリックで対応する処理画面へ遷移
直近の予定 ユーザーの直近の予定を表示 予定詳細の確認が可能

3. ダッシュボードコンテンツ

コンポーネント 説明 機能・振る舞い
ウェルカムバナー ユーザー名と重要なお知らせを表示 主要指標のハイライトを表示
KPI概要カード 主要指標をカード形式で表示 案件数・技術者数・稼働率・売上見込の表示
売上推移グラフ 月次売上の推移をグラフで表示 年間・半期・四半期の切り替えが可能
案件ステータス一覧 直近の案件状況を一覧表示 詳細ページへのリンクあり
マッチング候補 マッチングスコアの高い技術者を表示 候補者プロフィールへのリンクあり
通知パネル 最新の通知を表示 未読・既読の状態を視覚的に表示

画面動作仕様

1. ユーザーロール別表示項目

ユーザーロール 表示コンポーネント カスタマイズ
経営層 ヘッダー、通知パネル、KPI概要、業務進捗情報 KPI概要が拡張表示
営業担当者 ヘッダー、通知パネル、クイックアクション、業務進捗情報 案件関連のクイックアクションを優先表示
人事担当者 ヘッダー、通知パネル、クイックアクション、業務進捗情報 技術者関連のクイックアクションを優先表示
経理担当者 ヘッダー、通知パネル、クイックアクション、業務進捗情報 請求・支払関連のクイックアクションを優先表示
技術者 ヘッダー、通知パネル、クイックアクション、業務進捗情報 勤怠・工数入力へのクイックアクションを優先表示

2. 通知処理

操作 振る舞い
通知アイコンクリック ドロップダウンメニューで通知一覧を表示
通知リストアイテムクリック 対応する画面へ遷移し、既読状態に変更
全ての通知をクリックした場合 通知一覧画面へ遷移

3. メニュー操作

操作 振る舞い
メニューアイテムクリック 対応する機能画面へ遷移
未処理タスクアイテムクリック タスク処理画面へ直接遷移
予定アイテムクリック 予定詳細ポップアップを表示

4. KPI表示

操作 振る舞い
KPIカードクリック 対応する詳細レポート画面へ遷移
グラフ期間切り替え 選択した期間でグラフデータを再描画
グラフにマウスオーバー 詳細データをツールチップで表示

注意： ダッシュボードの表示内容はユーザー権限に応じて適切に制御されます。機密性の高い情報（売上情報、案件詳細情報など）は権限のあるユーザーにのみ表示されます。また、パフォーマンス最適化のため、グラフやデータの初期表示は直近6ヶ月分に制限されています。
