CNT-001: 契約一覧画面

画面概要

項目 内容
画面ID CNT-001
画面名 契約一覧
概要 全契約の一覧表示・検索・フィルタリングを行う画面
対象ユーザー 営業担当者、経理担当者
前画面 CMN-002: ダッシュボード
次画面 CNT-003: 契約詳細、CNT-002: 契約作成

画面構成要素

1. 検索・フィルター部分

項目ID 項目名 種類 説明
cnt_list_001 検索ボックス テキスト入力 契約名、契約ID、顧客名などでのフリーワード検索
cnt_list_002 契約種別 セレクトボックス（複数選択可） 契約種別によるフィルタリング（業務委託、派遣、請負など）
cnt_list_003 契約ステータス セレクトボックス（複数選択可） 契約ステータスによるフィルタリング（下書き、承認待ち、締結済み、終了など）
cnt_list_004 契約期間（開始） 日付範囲選択 契約開始日によるフィルタリング
cnt_list_005 契約期間（終了） 日付範囲選択 契約終了日によるフィルタリング
cnt_list_006 顧客名 セレクトボックス 顧客名によるフィルタリング
cnt_list_007 担当者 セレクトボックス 担当者によるフィルタリング
cnt_list_008 検索ボタン ボタン 検索条件を適用して結果を表示
cnt_list_009 クリアボタン ボタン 検索条件をクリア

2. 一覧表示部分

項目ID 項目名 種類 説明
cnt_list_010 契約一覧テーブル テーブル 検索条件に合致する契約の一覧
cnt_list_011 契約ID テキスト（リンク） システム内部で管理される契約の一意識別子。クリックで詳細画面へ
cnt_list_012 契約名 テキスト（リンク） 契約の名称。クリックで詳細画面へ
cnt_list_013 契約種別 テキスト 契約の種別（業務委託、派遣、請負など）
cnt_list_014 契約ステータス ラベル（色分け表示） 契約の現在のステータス
cnt_list_015 顧客名 テキスト 契約の相手先となる顧客企業名
cnt_list_016 契約期間 テキスト 契約の開始日～終了日
cnt_list_017 契約金額 数値（¥フォーマット） 契約の総額または月額
cnt_list_018 担当者 テキスト 契約の担当者名
cnt_list_019 最終更新日 日時 契約情報の最終更新日時
cnt_list_020 アクションボタン ボタングループ 詳細表示、編集、PDF表示などの操作ボタン
cnt_list_021 ページネーション ページ制御 複数ページにわたる結果のナビゲーション

3. 操作ボタン

項目ID 項目名 種類 説明
cnt_list_022 新規契約作成ボタン プライマリーボタン 新規契約作成画面への遷移
cnt_list_023 CSVエクスポートボタン セカンダリーボタン 表示中の契約一覧をCSV形式でエクスポート
cnt_list_024 CSVインポートボタン セカンダリーボタン CSV形式の契約情報をインポート

画面動作仕様

  1. 画面初期表示時は、デフォルトですべての有効な契約を表示する
  2. 検索条件を指定して検索ボタンをクリックすると、条件に合致する契約が一覧表示される
  3. 一覧の契約ID・契約名をクリックすると、該当の契約詳細画面（CNT-003）に遷移する
  4. アクションボタンの「詳細」をクリックすると、該当の契約詳細画面（CNT-003）に遷移する
  5. アクションボタンの「編集」をクリックすると、該当の契約を編集モードで開く
  6. アクションボタンの「PDF」をクリックすると、該当の契約書をPDF形式で表示（CNT-004）
  7. 「新規契約作成」ボタンをクリックすると、契約作成画面（CNT-002）に遷移する
  8. 「CSVエクスポート」ボタンをクリックすると、現在の検索結果がCSVファイルとしてダウンロードされる
  9. 「CSVインポート」ボタンをクリックすると、CSVインポート用のモーダルダイアログが表示される
  10. ページネーションを使用して、複数ページにわたる検索結果をナビゲートできる

注意： ユーザーの権限レベルによって、表示される操作ボタンや実行可能な操作が異なります。

  • 営業担当者：契約の作成・編集が可能
  • 経理担当者：契約の閲覧のみ可能、編集は不可
