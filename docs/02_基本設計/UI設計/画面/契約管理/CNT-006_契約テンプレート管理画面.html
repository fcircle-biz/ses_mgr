画面動作仕様

  1. 画面初期表示時は、利用可能なすべてのテンプレートが一覧表示される
  2. テンプレート名をクリックすると、そのテンプレートの詳細情報が表示される
  3. アクションボタンの「編集」をクリックすると、テンプレート編集モードになる
  4. アクションボタンの「複製」をクリックすると、現在のテンプレートを複製して新規テンプレートを作成する
  5. アクションボタンの「削除」をクリックすると、テンプレートを削除する
    • 確認ダイアログを表示してユーザーの意思を確認
    • 使用中のテンプレートは削除できない
  6. 「新規テンプレート作成」ボタンをクリックすると、新規テンプレート作成モードになる
  7. 編集モードでの「保存」ボタンクリック時
    • 入力内容のバリデーションを行い、問題がなければテンプレート情報を保存
    • 保存成功時は表示モードに戻り、更新された情報が表示される
    • 保存失敗時はエラーメッセージを表示
  8. 「プレビュー」ボタンをクリックすると、テンプレートのプレビューがモーダルウィンドウで表示される
  9. 「キャンセル」ボタンをクリックすると、編集内容を破棄して表示モードに戻る
  10. 「一覧に戻る」ボタンをクリックすると、契約一覧画面（CNT-001）に戻る

注意： この画面は営業管理者権限を持つユーザーのみアクセス可能です。テンプレートの編集は、システム全体の契約作成に影響するため、十分注意して行ってください。
