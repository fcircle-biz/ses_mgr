CNT-005: 電子署名管理画面

画面概要

項目 内容
画面ID CNT-005
画面名 電子署名管理
概要 電子署名の状況管理・履歴確認を行う画面
対象ユーザー 営業担当者
前画面 CNT-003: 契約詳細
次画面 CNT-003: 契約詳細

画面レイアウト

[電子署名管理画面のワイヤーフレーム]

主要な機能

  • 署名依頼の送信
  • 署名状況の管理・確認
  • 署名履歴の表示
  • 署名の再依頼・キャンセル
  • 署名済み文書の取得

画面動作仕様

  1. 画面初期表示時は、選択した契約の電子署名状況と履歴が表示される
  2. 「署名依頼」ボタンをクリックすると、署名依頼先のメールアドレスを入力するダイアログが表示される
  3. 署名依頼を送信すると、署名依頼メールが指定したアドレスに送信され、状況が「署名待ち」に更新される
  4. 「再依頼」ボタンをクリックすると、同じ署名者に対して署名依頼メールを再送信する
  5. 「キャンセル」ボタンをクリックすると、署名依頼をキャンセルする
    • 確認ダイアログを表示してユーザーの意思を確認
    • キャンセル成功時は状況が「キャンセル済み」に更新される
  6. 「署名済み文書ダウンロード」ボタンをクリックすると、すべての署名が完了した契約書をPDF形式でダウンロードする
  7. 「署名ログ表示」ボタンをクリックすると、詳細な署名処理ログを表示するモーダルダイアログが表示される
  8. 「戻る」ボタンをクリックすると、契約詳細画面（CNT-003）に戻る
