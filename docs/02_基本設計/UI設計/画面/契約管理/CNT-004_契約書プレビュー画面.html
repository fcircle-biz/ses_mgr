CNT-004: 契約書プレビュー画面

画面概要

項目 内容
画面ID CNT-004
画面名 契約書プレビュー
概要 生成された契約書のプレビュー表示・編集を行う画面
対象ユーザー 営業担当者
前画面 CNT-002: 契約作成、CNT-003: 契約詳細
次画面 CNT-003: 契約詳細

画面レイアウト

[契約書プレビュー画面のワイヤーフレーム]

主要な機能

  • 契約書のPDFプレビュー表示
  • 契約書内容の編集（テキスト置換、条項の追加・削除）
  • 契約書のエクスポート（PDF、Word形式）
  • 契約書の印刷

画面動作仕様

  1. 画面初期表示時は、契約情報から生成されたPDF契約書がプレビュー表示される
  2. 編集モードを選択すると、契約書のテキスト内容を編集できるようになる
  3. 「PDFダウンロード」ボタンをクリックすると、現在表示中の契約書がPDF形式でダウンロードされる
  4. 「Wordダウンロード」ボタンをクリックすると、現在表示中の契約書がWord形式でダウンロードされる
  5. 「印刷」ボタンをクリックすると、現在表示中の契約書が印刷ダイアログを表示して印刷できる
  6. 「保存して戻る」ボタンをクリックすると、編集内容が保存され契約詳細画面（CNT-003）に戻る
  7. 「戻る」ボタンをクリックすると、編集内容を保存せずに契約詳細画面（CNT-003）に戻る
    • 編集内容がある場合は確認ダイアログを表示
