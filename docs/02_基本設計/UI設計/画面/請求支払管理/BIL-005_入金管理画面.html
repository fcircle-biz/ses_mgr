画面動作仕様

  1. 画面初期表示時は、指定された請求書の情報と入金履歴が表示される
  2. 「入金日」のカレンダーコントロールから日付を選択可能
  3. 「入金金額」に金額を入力すると、入金後の未入金金額が自動計算される
  4. 「全額入金」ボタンをクリックすると、現在の未入金金額が「入金金額」フィールドに自動入力される
  5. 「入金登録」ボタンをクリックすると、入力内容の検証後、新しい入金情報が記録される
  6. 入金情報が登録されると、入金履歴テーブルに追加表示され、入金済金額と未入金金額が更新される
  7. 入金ステータスは登録された入金金額に基づいて自動的に更新される：
    • 入金がない場合：「未入金」
    • 一部入金がある場合：「一部入金」
    • 全額入金された場合：「入金済」
    • 支払期限を過ぎても未入金の場合：「遅延」
    • 入金額が請求額より少ない場合：「不足」
    • 入金額が請求額より多い場合：「過剰」
  8. 「ステータス変更」ボタンから入金ステータスを手動で変更することも可能
  9. 入金履歴テーブルのアクションボタンから個別の入金記録を編集または削除可能
  10. 「領収書発行」ボタンをクリックすると、選択した入金に対する領収書プレビューが表示され、印刷またはPDF保存が可能
  11. 「請求書詳細に戻る」ボタンをクリックすると、請求書詳細画面（BIL-003）に戻る

注意事項：

  • 入金情報の登録・編集は経理担当者と管理者のみが実行可能です。
  • 入金額が請求額を超える場合は、確認のメッセージが表示されます。
  • 一度登録した入金情報を削除する場合、削除理由の入力が必要です。
  • 入金ステータスの手動変更は監査ログに記録されます。
  • 領収書発行後に入金情報を変更した場合、領収書の再発行が必要になります。
