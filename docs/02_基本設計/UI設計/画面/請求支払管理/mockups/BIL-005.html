入金処理ヘルプ
入金ステータスについて
  • 未入金: 入金が記録されていない状態
  • 一部入金: 請求額の一部が入金された状態
  • 入金済: 請求額が全額入金された状態
  • 遅延: 支払期限日を過ぎても未入金の状態
  • 不足: 最終入金日が経過したが入金額が不足している状態
  • 過剰: 請求額より多く入金された状態
操作ガイド
  1. 入金日、金額、入金方法は必須項目です
  2. 「全額入金」ボタンを使うと、未入金額が自動入力されます
  3. 入金情報を登録すると、入金ステータスは自動更新されます
  4. 入金記録は後から編集・削除することも可能です
  5. 領収書は入金登録後に発行できます
