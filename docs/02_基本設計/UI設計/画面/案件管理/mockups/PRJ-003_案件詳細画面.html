案件概要
老朽化した基幹システムを刷新し、クラウド環境への移行を実施するプロジェクト。ユーザーインターフェース改善と業務効率化を主な目的として、最新技術を活用したシステム構築を行う。
詳細情報

現行システムは、20年前に開発されたクライアントサーバー型システムであり、保守性・拡張性の課題を抱えている。また、業務プロセスの変化に対応できておらず、多くの業務が手作業で補完されている状況である。

新システムでは以下の点を重視して開発を行う：

  • マイクロサービスアーキテクチャの採用による柔軟性と拡張性の確保
  • クラウドネイティブ環境への完全移行
  • モバイルデバイスからのアクセス対応
  • BI機能の強化によるデータ活用促進
  • APIによる外部システム連携の容易化

プロジェクトは複数フェーズに分けて実施し、段階的なリリースを予定している。
