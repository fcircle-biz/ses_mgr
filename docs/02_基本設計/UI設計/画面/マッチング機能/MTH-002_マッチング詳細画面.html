画面動作仕様

  1. 画面初期表示時は、マッチング検索結果から選択されたマッチングの詳細情報を表示する
  2. 「案件ID」「案件名」をクリックすると、該当の案件詳細画面（PRJ-003）に遷移する
  3. 「技術者ID」「技術者名」をクリックすると、該当の技術者詳細画面（ENG-003）に遷移する
  4. 「提案登録ボタン」をクリックすると、提案管理画面（MTH-003）に遷移し、現在のマッチング情報が引き継がれる
  5. 「スキルシート表示ボタン」をクリックすると、該当技術者のスキルシート管理画面（ENG-004）に遷移する
  6. 「マッチング結果保存ボタン」をクリックすると、保存形式を選択するポップアップが表示され、選択した形式でダウンロードされる
  7. 「次のマッチング結果ボタン」「前のマッチング結果ボタン」をクリックすると、検索結果リスト内の別のマッチング結果に切り替わる
  8. 「結果一覧へ戻るボタン」をクリックすると、マッチング検索画面（MTH-001）の検索結果表示部分に戻る

注意： マッチングスコアの詳細と比較情報は、マッチングアルゴリズムの設定によって異なります。

  • 要件比較部分では、要件を満たしている項目は緑色、部分的に満たしている項目は黄色、満たしていない項目は赤色で表示されます
  • 総合マッチングスコアが一定値（例：70点）以上の場合のみ「提案登録ボタン」が有効になります
