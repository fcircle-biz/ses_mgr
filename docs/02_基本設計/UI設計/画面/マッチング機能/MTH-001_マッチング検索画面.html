MTH-001: マッチング検索画面

画面概要

項目 内容
画面ID MTH-001
画面名 マッチング検索
概要 案件または技術者を起点に、条件指定してマッチング検索を行う画面
対象ユーザー 営業担当者、人事担当者
前画面 CMN-002: ダッシュボード、PRJ-003: 案件詳細、ENG-003: 技術者詳細
次画面 MTH-002: マッチング詳細

画面動作仕様

  1. 画面初期表示時は、画面遷移元によって以下のように動作する：
    • ダッシュボードからの遷移：空の検索フォームを表示
    • 案件詳細からの遷移：「案件起点検索」タブを選択し、該当案件を自動選択
    • 技術者詳細からの遷移：「技術者起点検索」タブを選択し、該当技術者を自動選択
  2. 「案件起点検索」タブと「技術者起点検索」タブの切り替えにより、表示される検索条件フォームが切り替わる
  3. 案件/技術者選択後、「条件自動反映」チェックボックスがONの場合は、各検索条件に自動的に対象の情報が設定される
  4. 「アルゴリズム選択」で「詳細マッチング」または「AIレコメンド」を選択すると、「重み付け設定」パネルが表示される
  5. 「検索ボタン」クリック時、入力された検索条件に基づきマッチング検索を実行し、結果プレビュー部分に表示する
  6. 検索結果の各行の「詳細表示ボタン」をクリックすると、該当のマッチングの詳細画面（MTH-002）に遷移する
  7. 「結果一括表示ボタン」をクリックすると、全マッチング結果を表示するマッチング詳細画面（MTH-002）に遷移する
  8. 「条件保存ボタン」をクリックすると、現在の検索条件に名前を付けて保存するモーダルダイアログが表示される

注意： マッチングアルゴリズムの選択と重み付け設定によって検索結果が大きく変わる可能性があります。

  • 基本マッチング：スキルと経験年数を主体とした基本的なマッチング
  • 詳細マッチング：より多くの要素を考慮した詳細なマッチング（重み付け調整可能）
  • AIレコメンド：過去の成約データなどを学習したAIによるレコメンデーション
