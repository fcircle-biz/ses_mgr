プロジェクト概要

大手アパレル企業のECサイトを、レガシーシステムからモダンなReactベースのシステムへ完全リニューアル。商品管理、在庫管理、注文処理、決済連携、会員管理などの機能を備えた大規模ECプラットフォームを構築。モバイルファーストの設計とパフォーマンス最適化を重視し、コンバージョン率の向上を実現。
