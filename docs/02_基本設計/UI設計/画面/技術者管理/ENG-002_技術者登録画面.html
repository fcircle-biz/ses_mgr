画面動作仕様

  1. 必須項目が入力されているかをバリデーションチェックする
    • 未入力の必須項目がある場合はエラーメッセージを表示
    • 入力形式や値の範囲が不正な場合はエラーメッセージを表示
  2. 所属区分で「社員」以外を選択した場合、所属会社の入力欄を必須として表示する
  3. 稼働状況で「稼働中」を選択した場合、対応可能日の入力欄は非表示とする
  4. 希望単価の下限が上限を超える値に設定された場合、エラーメッセージを表示する
  5. 「登録」ボタンをクリックした場合
    • 入力内容のバリデーションを行い、問題がなければ技術者情報を登録する
    • 登録成功時は技術者詳細画面（ENG-003）に遷移する
    • 登録失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  6. 「キャンセル」ボタンをクリックした場合
    • 入力内容を破棄して技術者一覧画面（ENG-001）に戻る
    • 入力内容がある場合は確認ダイアログを表示し、ユーザーの意思を確認する
