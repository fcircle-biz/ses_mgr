画面モード

この画面は以下の2つのモードを持ちます：

  1. 表示モード: 技術者の詳細情報を参照するためのモード（デフォルト）
  2. 編集モード: 技術者の詳細情報を編集するためのモード（人事担当者または本人のみ）

タブ構成

技術者詳細画面は以下のタブで構成されます：

  1. 基本情報: 技術者の基本的な情報（氏名、連絡先、状況など）
  2. スキル情報: 技術スキル、資格、経験など
  3. 稼働履歴: 過去の案件履歴
  4. マッチング履歴: 過去のマッチング履歴

主要な機能

  • 技術者の基本情報の表示・編集
  • 技術者のスキル情報の表示・編集
  • 稼働状況の変更
  • スキルシート管理画面への遷移
  • 過去の稼働履歴・案件履歴の表示
  • マッチング検索の実行

画面動作仕様

  1. 画面初期表示時は、表示モードで基本情報タブが選択された状態で表示
  2. 各タブをクリックすると、対応する情報が表示されるようタブ切り替えが行われる
  3. 「編集」ボタンをクリックすると編集モードに切り替わる（権限のある場合のみ表示）
  4. 編集モードでの「保存」ボタンクリック時
    • 入力内容のバリデーションを行い、問題がなければ技術者情報を更新する
    • 更新成功時は表示モードに戻り、更新された情報が表示される
    • 更新失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  5. 「キャンセル」ボタンをクリックすると、編集内容を破棄して表示モードに戻る
  6. 「スキルシート」ボタンをクリックすると、スキルシート管理画面（ENG-004）に遷移する
  7. 「マッチング検索」ボタンをクリックすると、この技術者の条件でマッチング検索画面（MTH-001）に遷移する
  8. 「一覧に戻る」ボタンをクリックすると、技術者一覧画面（ENG-001）に戻る

注意： 編集モードへの切り替えボタンは、編集権限を持つユーザー（人事担当者または本人）にのみ表示されます。
