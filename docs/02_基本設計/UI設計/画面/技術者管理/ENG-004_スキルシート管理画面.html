画面動作仕様

  1. 画面初期表示時は、デフォルトのテンプレートでスキルシートのプレビューを表示
  2. テンプレート選択ドロップダウンから別のテンプレートを選択すると、選択したテンプレートに応じたプレビューが表示される
  3. テンプレートサムネイルをクリックしても、対応するテンプレートが選択される
  4. 表示オプションのスイッチを切り替えると、プレビュー内容がリアルタイムで更新される
    • 写真表示オプションをオフにすると、技術者の写真が非表示になる
    • 連絡先表示オプションをオフにすると、連絡先情報が非表示になる
    • プロジェクト履歴表示オプションをオフにすると、プロジェクト履歴セクションが非表示になる
    • 資格表示オプションをオフにすると、資格情報セクションが非表示になる
  5. 「編集」ボタンをクリックすると編集モードに切り替わり、スキルシートの内容を編集できるようになる
  6. 編集モードでの「保存」ボタンクリック時
    • 入力内容のバリデーションを行い、問題がなければスキルシート情報を更新する
    • 更新成功時は表示モードに戻り、更新された情報でプレビューが表示される
    • 更新失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  7. 「PDFエクスポート」ボタンをクリックすると、現在のプレビュー内容がPDF形式でダウンロードされる
  8. 「Wordエクスポート」ボタンをクリックすると、現在のプレビュー内容がWord形式でダウンロードされる
  9. 「印刷」ボタンをクリックすると、ブラウザの印刷ダイアログが表示される
  10. 「戻る」ボタンをクリックすると、技術者詳細画面（ENG-003）に戻る

注意事項： スキルシートに含める個人情報は適切に管理し、クライアントに提出する際には必要に応じて連絡先などの情報を非表示にする設定を行ってください。個人情報保護の観点から、表示オプションを適切に設定した上でエクスポートすることが重要です。
