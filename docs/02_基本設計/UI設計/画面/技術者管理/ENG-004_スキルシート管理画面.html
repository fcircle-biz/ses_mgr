ENG-004: スキルシート管理画面

画面概要

項目 内容
画面ID ENG-004
画面名 スキルシート管理
概要 技術者のスキルシート作成・編集・出力を行う画面
対象ユーザー 人事担当者、本人
前画面 ENG-003: 技術者詳細
次画面 ENG-003: 技術者詳細

画面レイアウト

[スキルシート管理画面のワイヤーフレーム]

主要な機能

  • スキルシートのテンプレート選択
  • スキルシートのプレビュー表示
  • スキルシートの内容編集
  • スキルシートのPDF/Word形式での出力

画面動作仕様

  1. 画面初期表示時は、デフォルトのテンプレートでスキルシートのプレビューを表示
  2. テンプレート選択ドロップダウンから別のテンプレートを選択すると、選択したテンプレートに応じたプレビューが表示される
  3. 「編集」ボタンをクリックすると編集モードに切り替わり、スキルシートの内容を編集できるようになる
  4. 編集モードでの「保存」ボタンクリック時
    • 入力内容のバリデーションを行い、問題がなければスキルシート情報を更新する
    • 更新成功時は表示モードに戻り、更新された情報でプレビューが表示される
    • 更新失敗時はエラーメッセージを表示し、入力内容を保持したままの状態で再表示する
  5. 「PDFエクスポート」ボタンをクリックすると、現在のプレビュー内容がPDF形式でダウンロードされる
  6. 「Wordエクスポート」ボタンをクリックすると、現在のプレビュー内容がWord形式でダウンロードされる
  7. 「戻る」ボタンをクリックすると、技術者詳細画面（ENG-003）に戻る
