ADM-001: ユーザー管理画面

画面概要

項目 内容
画面ID ADM-001
画面名 ユーザー管理
概要 システムユーザーの登録・編集・無効化などを管理する画面
対象ユーザー システム管理者
前画面 CMN-002: ダッシュボード
次画面 ユーザー登録/編集モーダル（同画面内）

画面レイアウト

画面構成要素

1. 検索条件部

項目ID 項目名 種類 説明
user-search-001 ユーザーID テキスト入力 検索条件：ユーザーID（半角英数字、30文字以内）
user-search-002 ユーザー名 テキスト入力 検索条件：ユーザー名（100文字以内）
user-search-003 メールアドレス テキスト入力 検索条件：メールアドレス（メールアドレス形式、100文字以内）
user-search-004 ロール セレクトボックス 検索条件：ユーザーロール

user-search-006 検索ボタン ボタン 検索実行
user-search-007 クリアボタン ボタン 検索条件クリア

user-list-004 ユーザー名 テキスト表示（ソート可） 一覧表示：ユーザー名
user-list-005 メールアドレス テキスト表示 一覧表示：メールアドレス
user-list-006 ロール テキスト表示（ソート可） 一覧表示：ユーザーロール
user-list-007 所属部門 テキスト表示 一覧表示：所属部門
user-list-008 ステータス アイコン表示（ソート可） 一覧表示：アカウント状態（有効/無効/ロック中）

3. ユーザー登録/編集モーダル

項目ID 項目名 種類 説明
user-edit-001 ユーザーID テキスト入力 ユーザーID（半角英数字、8〜30文字）（編集時は読み取り専用）
user-edit-002 氏名 テキスト入力 ユーザーの氏名（100文字以内）（必須）
user-edit-003 メールアドレス テキスト入力 ユーザーのメールアドレス（メールアドレス形式、100文字以内）（必須）
user-edit-004 パスワード パスワード入力 新規登録時のみ必須（半角英数記号、12〜30文字）
user-edit-005 ロール マルチセレクト ユーザーに付与するロール（複数選択可）（必須）
user-edit-006 所属部門 セレクトボックス ユーザーの所属部門（必須）
user-edit-007 電話番号 テキスト入力 ユーザーの連絡先電話番号（電話番号形式）
user-edit-008 多要素認証 チェックボックス 多要素認証の有効/無効
user-edit-009 アカウントロック チェックボックス アカウントのロック状態（編集時のみ表示）
user-edit-010 保存ボタン ボタン ユーザー情報の保存
user-edit-011 キャンセルボタン ボタン 編集のキャンセル

画面動作仕様

  1. 検索機能
    1. 検索条件を入力し、検索ボタンをクリックする
    2. 入力された条件に合致するユーザーを検索し、一覧に表示する
    3. 該当するユーザーがない場合は「該当するユーザーがありません」というメッセージを表示する
    4. クリアボタンをクリックすると、すべての検索条件をクリアする
  2. 一覧表示機能
    1. ユーザー情報を一覧形式で表示する
    2. 各カラムヘッダーをクリックすることで、そのカラムでソートができる
    3. 一覧の各行のチェックボックスで複数選択し、一括操作が可能
    4. ページングコントロールで表示ページの切り替えが可能
  3. ユーザー登録機能
    1. 新規登録ボタンをクリックすると、ユーザー登録モーダルが表示される
    2. 必要情報を入力し、保存ボタンをクリックする
    3. 入力値のバリデーションを行う
      • 必須項目が未入力の場合はエラーメッセージを表示
      • 入力形式が不正な場合はエラーメッセージを表示
      • ユーザーIDが既に使用されている場合はエラーメッセージを表示
    4. 登録成功時は一覧を更新し、成功メッセージを表示する
    5. 登録失敗時はエラーメッセージを表示する
  4. ユーザー編集機能
    1. 一覧の操作カラムから編集ボタンをクリックすると、ユーザー編集モーダルが表示される
    2. 現在の情報が初期表示され、編集可能（ただしユーザーIDは編集不可）
    3. 情報を編集し、保存ボタンをクリックする
    4. 入力値のバリデーションを行う
    5. 更新成功時は一覧を更新し、成功メッセージを表示する
    6. 更新失敗時はエラーメッセージを表示する
  5. アカウント状態管理機能
    1. 一覧の操作カラムから有効化/無効化ボタンをクリックすると、アカウント状態を変更する
    2. 複数アカウントを選択して一括操作ドロップダウンから有効化/無効化/ロック解除を選択することも可能
    3. 操作実行前に確認ダイアログを表示する
    4. 操作成功時は一覧を更新し、成功メッセージを表示する
  6. パスワードリセット機能
    1. 一覧の操作カラムからパスワードリセットボタンをクリックする
    2. 確認ダイアログが表示される
    3. 確認後、仮パスワードを生成し、ユーザーのメールアドレスに送信する
    4. 操作成功時は成功メッセージを表示する

注意：

  • ユーザー管理画面はシステム管理者ロールを持つユーザーのみアクセス可能
  • 自分自身のアカウントを無効化することはできない（エラーメッセージを表示）
  • 最後のシステム管理者アカウントを無効化することはできない（エラーメッセージを表示）
  • ユーザーの削除機能は提供せず、無効化のみとする（監査対応）
  • パスワードは画面上では表示されず、新規登録時のみ設定可能。編集時はパスワードリセット機能を使用
  • アカウントロックは連続ログイン失敗によって自動的に設定され、管理者が解除可能
