画面動作仕様

マスタデータ表示機能

  1. 画面初期表示時は、組織系マスタのタブが選択された状態で表示される
  2. マスタ分類タブを選択すると、対応するマスタ一覧が左側に表示される
  3. マスタ一覧から特定のマスタを選択すると、そのマスタデータの一覧が右側に表示される
  4. 検索ボックスに文字を入力すると、リアルタイムでマスタデータがフィルタリングされる

マスタデータ新規登録機能

  1. 新規登録ボタンをクリックすると、マスタデータ編集モーダルが表示される
  2. 必要情報を入力し、保存ボタンをクリックする
  3. 入力値のバリデーションを行う
    • 必須項目が未入力の場合はエラーメッセージを表示
    • 入力形式が不正な場合はエラーメッセージを表示
    • コードが既に使用されている場合はエラーメッセージを表示
  4. 登録成功時は一覧を更新し、成功メッセージを表示する
  5. 登録失敗時はエラーメッセージを表示する

マスタデータ編集機能

  1. 一覧の操作カラムから編集ボタンをクリックすると、マスタデータ編集モーダルが表示される
  2. 現在の情報が初期表示され、編集可能（ただしコードは編集不可）
  3. 情報を編集し、保存ボタンをクリックする
  4. 入力値のバリデーションを行う
  5. 更新成功時は一覧を更新し、成功メッセージを表示する
  6. 更新失敗時はエラーメッセージを表示する

マスタデータ削除機能

  1. 一覧の操作カラムから削除ボタンをクリックする
  2. 確認ダイアログが表示される
  3. 確認後、マスタデータの削除処理を実行する
  4. 関連データで参照されている場合は警告メッセージを表示し、削除を中止する
  5. 削除成功時は一覧を更新し、成功メッセージを表示する

有効/無効切り替え機能

  1. 一覧の操作カラムから有効化/無効化ボタンをクリックする
  2. 確認ダイアログが表示される
  3. 確認後、マスタデータの有効/無効状態を切り替える
  4. 関連データで使用中のマスタを無効化する場合は警告メッセージを表示する
  5. 処理成功時は一覧を更新し、成功メッセージを表示する

一括インポート機能

  1. 一括インポートボタンをクリックすると、インポートモーダルが表示される
  2. 必要に応じてテンプレートをダウンロードし、CSVファイルを準備する
  3. CSVファイルを選択し、インポートモードを指定して実行ボタンをクリックする
  4. ファイル形式のバリデーションを行い、不正な場合はエラーメッセージを表示する
  5. インポート処理中はプログレスバーを表示する
  6. インポート結果をサマリー表示する（成功件数、エラー件数など）
  7. エラーがある場合はエラー内容をダウンロード可能にする

一括エクスポート機能

  1. 一括エクスポートボタンをクリックする
  2. 現在表示しているマスタデータをCSVファイルにエクスポートする
  3. ファイルのダウンロードが開始される

マスタデータ分類と種類

組織系マスタ

  • 部門マスタ：会社の組織構造
  • 役職マスタ：社内の役職
  • 雇用形態マスタ：正社員、契約社員、派遣社員など
  • 取引先分類マスタ：顧客、パートナー、仕入先など

業務系マスタ

  • スキル分類マスタ：スキルの大分類（言語、DB、フレームワークなど）
  • スキル詳細マスタ：具体的なスキル項目
  • 案件ステータスマスタ：案件の状態（募集中、契約中、完了など）
  • 案件種別マスタ：SES、受託開発、コンサルなど
  • 稼働形態マスタ：常駐、リモート、ハイブリッドなど
  • 契約種別マスタ：準委任、請負など
  • 請求サイクルマスタ：月次、週次、完了時など
  • 単価種別マスタ：時給、日給、月給など

システム系マスタ

  • システム設定マスタ：システム全体の設定値
  • 通知テンプレートマスタ：メール通知などのテンプレート
  • 帳票テンプレートマスタ：各種帳票のテンプレート
  • エラーコードマスタ：システムエラーコードと説明

特記事項

  • マスタ管理画面はシステム管理者ロールを持つユーザーのみアクセス可能
  • システム運用に必須のマスタデータは削除できない（無効化のみ可能）
  • マスタデータによって表示・編集項目が異なる（動的フォーム）
  • マスタデータの変更履歴はシステムログに記録される
  • 一度に表示されるマスタデータは最大100件とし、それ以上はページングする
  • 各マスタの相互関連性を考慮し、関連するマスタ間の整合性を保つ必要がある
