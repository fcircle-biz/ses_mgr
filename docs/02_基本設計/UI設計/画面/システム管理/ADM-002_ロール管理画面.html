画面動作仕様

ロール一覧表示機能

  1. システムに登録されているロール情報を一覧形式で表示する
  2. 各カラムヘッダーをクリックすることで、そのカラムでソートができる
  3. ページングコントロールで表示ページの切り替えが可能

ロール新規作成機能

  1. 新規ロール作成ボタンをクリックすると、ロール登録モーダルが表示される
  2. 基本情報タブでロールの基本情報を入力する
  3. 権限設定タブで機能ごとのアクセス権限を設定する
  4. 保存ボタンをクリックすると、入力値のバリデーションを行う
    • 必須項目が未入力の場合はエラーメッセージを表示
    • 入力形式が不正な場合はエラーメッセージを表示
    • ロールIDが既に使用されている場合はエラーメッセージを表示
  5. 登録成功時は一覧を更新し、成功メッセージを表示する
  6. 登録失敗時はエラーメッセージを表示する

ロール編集機能

  1. 一覧の操作カラムから編集ボタンをクリックすると、ロール編集モーダルが表示される
  2. 現在の情報が初期表示され、編集可能（ただしロールIDは編集不可）
  3. 基本情報タブと権限設定タブの情報を編集する
  4. 保存ボタンをクリックすると、入力値のバリデーションを行う
  5. 更新成功時は一覧を更新し、成功メッセージを表示する
  6. 更新失敗時はエラーメッセージを表示する

ロール削除機能

  1. 一覧の操作カラムから削除ボタンをクリックする
  2. 確認ダイアログが表示される
  3. 確認後、ロールの削除処理を実行する
  4. 削除成功時は一覧を更新し、成功メッセージを表示する
  5. 削除失敗時はエラーメッセージを表示する

ロール複製機能

  1. 一覧の操作カラムから複製ボタンをクリックする
  2. 選択したロールの情報を元に、新規ロール作成モーダルが表示される
  3. ロールIDには接頭辞「COPY_」が自動的に付与される
  4. その他の情報は元のロールと同じ値が初期表示される
  5. 情報を編集し、保存ボタンをクリックする

特記事項

  • ロール管理画面はシステム管理者ロールを持つユーザーのみアクセス可能
  • システム運用に最低限必要なシステム管理者ロールは削除できない（エラーメッセージを表示）
  • ユーザーが割り当てられているロールを削除する場合は警告メッセージを表示する
  • ロールに設定可能な権限は機能グループごとに階層化されて表示される
  • 権限設定では親グループのチェックボックスを選択すると、すべての子グループも選択される
  • システム区分のロールはシステム管理機能に関する権限を設定可能
  • 業務区分のロールは各業務機能（案件管理、技術者管理など）に関する権限を設定可能
