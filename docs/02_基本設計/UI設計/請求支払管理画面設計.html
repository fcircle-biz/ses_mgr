SES業務システム 請求・支払管理画面設計

画面一覧

請求・支払管理機能では、以下の画面を提供します。各画面の詳細設計は、リンク先のドキュメントを参照してください。

  • BIL-001 請求一覧画面 顧客に対する請求情報の一覧表示・検索・フィルタリングを行う画面
  • BIL-002 請求書作成画面 新規請求書情報を作成する画面（工数データと連携、自動計算）
  • BIL-003 請求書詳細画面 請求書の詳細情報表示・編集を行う画面
  • BIL-004 請求書プレビュー画面 生成された請求書のプレビュー表示画面
  • BIL-005 入金管理画面 請求に対する入金状況の管理画面
  • BIL-006 支払一覧画面 パートナーや技術者への支払情報の一覧表示・検索・フィルタリングを行う画面
  • BIL-007 支払登録画面 新規支払情報を登録する画面
  • BIL-008 支払詳細画面 支払情報の詳細表示・編集を行う画面

機能概要

機能 説明
請求一覧 顧客に対する請求情報の一覧表示・検索・フィルタリング機能
請求書作成 新規請求書情報の作成機能（工数データと連携、自動計算）
請求書詳細 請求書の詳細情報表示・編集機能
請求書プレビュー 生成された請求書のプレビュー表示機能
入金管理 請求に対する入金状況の管理機能
支払一覧 パートナーや技術者への支払情報の一覧表示・検索・フィルタリング機能
支払登録 新規支払情報の登録機能
支払詳細 支払情報の詳細表示・編集機能
支払承認フロー 支払情報の承認ワークフロー機能

注意： ユーザーの権限レベルによって、表示される操作ボタンや実行可能な操作が異なります。

  • 一般ユーザー：担当する請求・支払の閲覧のみ可能
  • 経理担当者：全ての請求・支払の閲覧・編集が可能
  • 管理者：全ての請求・支払の閲覧・編集・承認が可能
  • 経営層：全ての請求・支払の閲覧が可能だが、編集は不可
