SES業務システム 画面遷移図

1. 画面遷移図について

本ドキュメントは、SES業務システムにおける画面間の遷移関係を図示したものです。 各機能領域ごとの画面遷移と、領域間の連携を示しています。

凡例

  • 矢印：画面遷移の方向（一方向または双方向）
  • 箱：各画面（画面ID-画面名）
  • 破線の枠：機能モジュール
  • 矢印のラベル：遷移の契機・条件

2. 全体画面遷移図

4. 補足事項

上記の画面遷移図は、主要な画面間の遷移を示したものです。実際のシステムでは、より詳細な遷移やポップアップ・モーダルダイアログからの遷移も存在します。 また、各画面からグローバルメニュー（CMN-003）を経由して、他の機能へのアクセスも可能です。

各画面の詳細な仕様については、対応する画面設計ドキュメントをご参照ください。
