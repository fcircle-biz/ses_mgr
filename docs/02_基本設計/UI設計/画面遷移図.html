画面遷移図はdraw.ioで作成されており、.drawioファイルとして保存されています。

画面遷移図の見方

画面遷移図では以下の要素で表現しています：

  • ノード（四角形）：各画面を表しています。ID・名称で識別されています。
  • エッジ（矢印）：画面間の遷移を表しています。
  • ラベル：画面遷移の契機（ボタンクリック、リンク選択など）を示しています。
  • 色分け：機能領域ごとに色分けされており、各機能の画面を識別しやすくしています。

画面一覧と合わせてご参照ください。
