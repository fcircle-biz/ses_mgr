SES業務システム システム管理画面設計

システム管理画面一覧

SES業務システムのシステム管理画面設計に関する詳細です。各画面の詳細仕様は、画面IDのリンクから確認できます。

  • ADM-001: ユーザー管理画面
    システムユーザーの登録・編集・無効化などを管理する画面
  • ADM-002: ロール管理画面
    ユーザーロールと権限の設定を管理する画面
  • ADM-003: マスタ管理画面
    システムで使用する各種マスタデータを管理する画面
  • ADM-004: ログ管理画面
    システムログの閲覧・検索・エクスポートを行う画面
  • ADM-005: バッチ管理画面
    バッチ処理の実行・スケジュール設定・履歴確認を行う画面

機能概要

機能 説明
ユーザー管理 システムユーザーの登録・編集・無効化などを管理する機能
ロール管理 ユーザーロールと権限の設定を管理する機能
マスタ管理 システムで使用する各種マスタデータを管理する機能
ログ管理 システムログの閲覧・検索・エクスポートを行う機能
バッチ管理 バッチ処理の実行・スケジュール設定・履歴確認を行う機能

注意： システム管理機能はシステム管理者ロールを持つユーザーのみがアクセスできます。

  • ユーザー管理：ユーザーアカウントのライフサイクル管理を行います
  • ロール管理：システム全体で使用する権限とロールの設定を行います
  • マスタ管理：システム運用に必要な各種マスタデータのメンテナンスを行います
  • ログ管理：システム操作ログの検索・分析・エクスポートを行います
  • バッチ管理：定期実行・手動実行のバッチ処理を管理します
