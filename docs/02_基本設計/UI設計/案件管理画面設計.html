SES業務システム 案件管理画面設計

画面一覧

案件管理機能では、以下の画面を提供します。各画面の詳細設計は、リンク先のドキュメントを参照してください。

  • PRJ-001 案件一覧画面 全案件の一覧表示・検索・フィルタリングを行う画面
  • PRJ-002 案件登録画面 新規案件情報を登録する画面
  • PRJ-003 案件詳細画面 案件の詳細情報表示・編集を行う画面

機能概要

機能 説明
案件一覧 全案件の一覧表示・検索・フィルタリング機能
案件登録 新規案件情報の登録機能
案件詳細 案件の詳細情報表示・編集機能
案件ステータス管理 案件のライフサイクルに沿ったステータス管理機能
案件マッチング 案件と技術者のマッチング機能との連携

注意： ユーザーの権限レベルによって、表示される操作ボタンや実行可能な操作が異なります。

  • 営業担当者：自分が担当する案件のみ編集可能
  • 営業管理者：全案件の閲覧・編集が可能
  • 経営層：全案件の閲覧が可能だが、編集は不可
